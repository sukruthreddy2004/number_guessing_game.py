# Number Guessing Game (Python)

A simple Python number guessing game built using loops and random numbers.

## How it works
- Program selects a random number between 1 and 100
- User guesses until they find the correct number
- App gives hints: **Too low** / **Too high**
- Shows how many attempts were made

## Concepts used
- Python basics (input, variables)
- Loops (`while`)
- Conditions (`if / elif / else`)
- `random.randint()` for random numbers

## How to run
1. Clone this repository or download the script.
2. Make sure Python is installed.
3. Run this command:

   ```bash
   python main.py
